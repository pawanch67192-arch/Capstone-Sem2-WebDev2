#🎮 Gamified Task Manager#

A full-stack Gamified Task Manager built with React that transforms daily productivity into an engaging experience. Users can manage tasks while earning XP, leveling up, maintaining streaks, and unlocking achievements.

#🚀 Project Overview#

This application combines productivity and gamification to motivate users to complete daily tasks. Instead of a traditional to-do list, users are rewarded with points, levels, badges, and streaks—making task management fun and addictive.

#✨ Features#

✅ Task Management
Add, edit, and delete tasks
Set task priority:
Easy → +10 XP
Medium → +20 XP
Hard → +50 XP
Assign due dates and reminders

🎯 XP System
Earn XP based on task difficulty
XP contributes to user progression
Instant feedback on task completion

🧩 Level System
Levels increase based on accumulated XP
Dynamic leveling formula
Progress bar showing XP needed for next level

🔥 Streak System
Daily task completion streak tracking
Streak resets if a day is missed
Visual streak indicators to boost motivation

🏅 Badges & Achievements

Unlock achievements like:

🎉 First Task Completed
🔥 7-Day Streak
💯 100 Tasks Completed

Badges are stored in the user profile.

📊 Dashboard
Total XP overview
Current level display
Completed tasks count
Weekly performance analytics (charts)

🔐 Authentication
User Signup/Login system
Secure data storage
Personalized progress tracking

#🖥️ Tech Stack#

Frontend
⚛️ React (Hooks + Functional Components)
🔀 React Router
🎨 Tailwind CSS / Material UI

Backend (Choose One)
🌐 Node.js + Express
🔥 Firebase (alternative backend)

Database
🍃 MongoDB (for Node.js backend)


#🌟 Advanced Features:#

🌙 Dark Mode
🎮 Drag-and-drop task reordering
🔊 Sound effects & animations on XP gain
🏆 Leaderboard system
📱 Offline support using local storage sync
